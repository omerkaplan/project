*As of March 2026, this page describes how I use AI on this site (and somewhat in general)*

---

All the text on the website was written by me, I've never asked AI to write as me to pretend to be me but from what I've seen even if I did AI won't be able to capture my voice, structure and cadence. 

I am using AI to find better flow for certain sentences or discovering new vocabulary to use as I write, but I never let it go rogue and write entire paragraphs or come up with ideas. Gives me the ick when I sense it elsewhere, so I'm deliberately avoiding it. 

This site is built on Pelican as a platform and using Bootstrap+SASS for frontend - I'm not sure these are the best choices but those are the ones I feel comfortable tinkering with. I use Claude Code to mostly style and mess around with the grid, which I can do but at this point seems redundant.
